# Claude Code Web UI

基于 [@siteboon/claude-code-ui](https://github.com/siteboon/claudecodeui) 的 Claude Code 网页界面封装，提供浏览器端的 Claude Code 交互体验。

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

或直接双击 `start.bat`。

启动后访问 **http://localhost:3001**，默认账号 `admin / admin123`。

## 项目结构

```
claude-code-webui/
├── package.json      # 项目配置
├── start.bat         # Windows 一键启动脚本
├── project/          # 工作区目录
└── node_modules/     # 依赖包
```
